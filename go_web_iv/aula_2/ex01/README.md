# Exercício 1 - Documentação

É necessário implementar a documentação do projeto usando swagger, todos os
endpoints documentados devem estar nele. Para conseguir isso, você deve levar
em consideração:

1. Usar o pacote swaggo.
2. Gerar as anotações na função principal.
3. Fazer anotações nos controladores.
4. Desenvolver o pacote de documentos com swaggo com base nas anotações.
5. Adicionar um endpoint usando GET para visualizar a documentação gerada.
